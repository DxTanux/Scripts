## 写在前面

建立这个项目，我感觉是一件很有意义的事——尽管开始得很迟（高考假后才动手 >︿<）。

这一年，我确实折腾懂了不少东西。如果"嘉豪"是一个形容词，那这台电脑已经很**"嘉豪"**了。记得刚接手她的时候：桌面路径是乱的，环境变量是乱的，360 的"遗体"深埋在文件系统最深处……像一片大海，有垃圾，也有宝藏。我在旁边不断拾起贝壳，擦拭，储藏，然后把沙滩打扫干净。

**给岁月以文明。**

后来，我和她一起进步：OpenCore、OpenCode、QwenPaw、Ollama、Mac26Tahoe、Geek、Wallpaper、Nexus、Git、Ubuntu……我们也失败过很多次，也被老师抓过很多次（靠 ╰（‵□′）╯，习惯了）。

如果说我能给你们留下什么最好的东西，大概就是这套 **Win11 + Ubuntu 双系统**（可以逃过希沃集控的监控 👀），以及一个还算不错的开发环境。

> ⚠️ **别问为什么没时政**，问就是重装系统时把 C2G2 连接凭据搞没了。

> 📸 **千万不要在中午玩电脑！** 被抓会被值周老师拍照。别问我怎么知道的 QAQ。

> ⚠️ **我们假设看这个的是人类** 因该会用Github吧?

### 📑 作业 PPT 自动化

我们每天作业的 PPT 会自动套用模板样式，并且**旧版本会自动存档**，不用担心被覆盖。

| 项目 | 路径 |
|------|------|
| 🛠️ **脚本源码/程序** | `D:\Github\Scripts\` |
| 📁 **模板 & 作业存档** | `E:\学科资料\###\` |

- 模板可以直接替换/编辑，放新模板到对应目录即可生效
- 脚本已打包成 `.exe`，如果你会逆向/解包可以拿去改；当然，**更推荐你自己重写一个**

---

### ⏰ Classland（课程表 + 脚本触发中心）

**Classland** 不仅仅是一个顶部课程表显示工具，它更是很多自动化脚本的**触发器与调度中心**，功能非常强大。

- ✅ 时间表已经帮你们配置好了
- 💡 **强烈建议深入研究**，你可以用它触发更多自定义脚本
