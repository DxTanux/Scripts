# 🎯 ClassPC-Remander
> 电教日志
## 📌 项目介绍
**ClassPC-Remander** 是记录电教员工作,云端保存电教员的一些东西,和分享一些有趣的事情的一个项目
---
## 💡 项目意义
| 层面 | 说明 |
|------|------|
| 🎨 **用户体验** | 告别原生 Windows 的简陋界面，让电脑更美观、更易用 |
| ⏰ **效率提升** | 自动化任务减少重复劳动，让师生专注于教学和学习 |
| 👨‍💻 **技术传承** | 完整的文档和代码注释，让下一任电教员快速上手 |
| 🔧 **可维护性** | 模块化设计便于调试、扩展和版本管理 |
| 📚 **学习成长** | 通过实际项目锻炼编程、系统管理和文档编写能力 |
---
## 🚀 项目内容
### 🔹 功能模块
#### 1️⃣ 界面美化 (`ThemeMaster/`)
- 任务栏透明效果
- 开始菜单样式定制
- 桌面图标整理与美化
- 右键菜单精简
- 系统主题色自定义
#### 2️⃣ 每日课表 (`Schedule/`)
- 自动获取当日课程安排
- 桌面Widget实时显示
- 课程变动提醒通知
- 支持手动导入/自动同步
#### 3️⃣ 自动化作业 (`HomeworkBot/`)
- 作业提交自动检测
- 截止日期倒计时提醒
- 作业文件夹自动归类
- 作业完成状态可视化
#### 4️⃣ 壁纸轮换 (`WallpaperChanger/`)
- 支持本地壁纸库轮播
- 多媒体壁纸支持（图片/视频）
- 按课程/时间智能切换壁纸
- 一键恢复默认设置
#### 5️⃣ 系统工具 (`Tools/`)
- 软件静默安装脚本
- 开机自启管理
- 一键还原/备份设置
- 故障诊断日志收集
